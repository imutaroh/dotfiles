# Neovim キーマップ チートシート

> `<leader>` = スペースキー（AstroNvim デフォルト）

---

## ファイル・プロジェクト操作

| キー | 動作 | 備考 |
|------|------|------|
| `<leader>ff` | ファイル検索（Telescope） | プロジェクト内のファイルをあいまい検索 |
| `<leader>fg` | 全文検索（Live grep） | プロジェクト内の文字列を検索 |
| `<leader>fb` | バッファ一覧 | 開いているファイルの一覧 |
| `<leader>fh` | ヘルプ検索 | Neovim のドキュメントを検索 |
| `<leader>ft` | TODO 一覧 | プロジェクト内の TODO/FIXME/NOTE を一覧表示 |
| `<leader>fp` | プロジェクト切替 | .git があるディレクトリを一覧表示して切替 |
| `<leader>fr` | 最近のファイル | 最近開いたファイル一覧 |

---

## バッファ（タブ）操作

| キー | 動作 | 備考 |
|------|------|------|
| `<Tab>` | 次のバッファへ | ブラウザのタブ移動と同じ感覚（カスタム） |
| `<S-Tab>` | 前のバッファへ | カスタム |
| `<leader>c` | バッファを閉じる | |
| `<leader>C` | バッファを強制閉じ | 変更を破棄して閉じる |

## Git 操作

| キー | 動作 | 備考 |
|------|------|------|
| `<leader>gg` | LazyGit を開く | Git の TUI クライアント。コミット・差分確認など何でもできる |
| `<leader>gf` | 現在ファイルの Git 履歴（LazyGit） | |
| `<leader>gd` | Diffview を開く | git diff を横並びパネルで確認 |
| `<leader>gD` | Diffview を閉じる | |
| `<leader>gh` | 現在ファイルの Git 履歴（Diffview） | |
| `gh` | 次の変更ハンク（行）へ | gitsigns（カスタム） |
| `gH` | 前の変更ハンク（行）へ | gitsigns（カスタム） |

---

## LSP（コード補助）

| キー | 動作 | 備考 |
|------|------|------|
| `gd` | 定義にジャンプ | 関数や変数の定義元へ飛ぶ |
| `gD` | 宣言にジャンプ | |
| `gr` | 参照一覧 | その関数が使われている箇所を一覧表示 |
| `gI` | 実装にジャンプ | インターフェースの実装先へ |
| `K` | ホバードキュメント | カーソル下のシンボルのドキュメントを表示 |
| `<leader>la` | コードアクション | Quick fix, import 追加 など |
| `<leader>lr` | リネーム | プロジェクト全体で一括リネーム |
| `<leader>lf` | フォーマット | |
| `<leader>ld` | 診断を表示 | カーソル下のエラー詳細 |
| `]d` / `[d` | 次/前の診断へ | エラー・警告を順番に移動 |

## テキスト操作（surround）

| キー | 動作 | 例 |
|------|------|----|
| `ys{motion}{文字}` | テキストを囲む | `ysiw"` → `word` → `"word"` |
| `ys{motion}<タグ>` | HTML タグで囲む | `ysiw<div>` → `<div>word</div>` |
| `cs{旧}{新}` | 囲みを変更 | `cs"'` → `"hello"` → `'hello'` |
| `ds{文字}` | 囲みを削除 | `ds"` → `"hello"` → `hello` |
| `dst` | HTML タグを削除 | `dst` → `<div>word</div>` → `word` |
