# Neovim チートシート

AstroNvim v5 ベースの設定に基づいたキーマップ一覧。  
Leader キー = `Space`

---

## 起動

```bash
gdev              # Ghostty を左右分割して nvim + Claude Code を起動（デフォルト: ~/dotfiles）
gdev ~/my-project # プロジェクトを指定して起動
nvim              # 単体で起動
```

---

## モード

| キー | モード | 説明 |
|------|--------|------|
| `i` | → Insert | カーソル位置から入力 |
| `a` | → Insert | カーソルの次から入力 |
| `o` | → Insert | 下に新しい行を追加して入力 |
| `O` | → Insert | 上に新しい行を追加して入力 |
| `v` | → Visual | 文字単位で選択 |
| `V` | → Visual | 行単位で選択 |
| `Esc` | → Normal | どのモードからでも Normal に戻る |

---

## 移動（Normal モード）

### 基本

| キー | 動作 |
|------|------|
| `h / j / k / l` | 左 / 下 / 上 / 右 |
| `w` | 次の単語の先頭へ |
| `b` | 前の単語の先頭へ |
| `e` | 次の単語の末尾へ |
| `0` | 行頭へ |
| `$` | 行末へ |
| `gg` | ファイルの先頭へ |
| `G` | ファイルの末尾へ |
| `{数字}G` | 指定行へ（例: `10G` → 10行目） |

### スクロール

| キー | 動作 |
|------|------|
| `Ctrl+d` | 半ページ下へ |
| `Ctrl+u` | 半ページ上へ |
| `Ctrl+f` | 1ページ下へ |
| `Ctrl+b` | 1ページ上へ |
| `zz` | カーソル行を画面中央に |

---

## 編集（Normal モード）

### 削除

| キー | 動作 |
|------|------|
| `x` | カーソルの文字を削除 |
| `dd` | 行を削除（クリップボードに入る） |
| `dw` | カーソルから次の単語まで削除 |
| `d$` | カーソルから行末まで削除 |
| `diw` | カーソル上の単語を削除 |

### 変更

| キー | 動作 |
|------|------|
| `r` | カーソルの1文字を置換 |
| `ciw` | カーソル上の単語を変更（Insert に入る） |
| `ci"` | `"..."` の中身を変更 |
| `ci(` | `(...)` の中身を変更 |
| `cc` | 行全体を変更 |
| `C` | カーソルから行末まで変更 |

### コピー＆ペースト

| キー | 動作 |
|------|------|
| `yy` | 行をコピー |
| `yw` | 単語をコピー |
| `yiw` | カーソル上の単語をコピー |
| `p` | カーソルの後にペースト |
| `P` | カーソルの前にペースト |

### Undo / Redo

| キー | 動作 |
|------|------|
| `u` | Undo |
| `Ctrl+r` | Redo |

---

## バッファ（タブ）

| キー | 動作 |
|------|------|
| `Tab` | 次のバッファへ |
| `Shift+Tab` | 前のバッファへ |
| `Space + b + d` | 現在のバッファを閉じる |

---

## AstroNvim キーマップ（Space リーダー）

### ファイル検索

| キー | 動作 |
|------|------|
| `Space + f + f` | ファイル名で検索 |
| `Space + f + g` | ファイル内テキストを全文検索（Grep） |
| `Space + f + r` | 最近開いたファイルを検索 |
| `Space + f + b` | 開いているバッファを検索 |

### Git

| キー | 動作 |
|------|------|
| `Space + g + g` | LazyGit を起動 |
| `Space + g + s` | Git ステータスを表示 |
| `]g` | 次の Git 差分へ |
| `[g` | 前の Git 差分へ |

### LSP（コード補助）

| キー | 動作 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照を一覧表示 |
| `K` | ドキュメントをホバー表示 |
| `Space + l + r` | シンボルをリネーム |
| `Space + l + a` | コードアクションを表示 |
| `Space + l + f` | ファイルをフォーマット |

### ウィンドウ分割

| キー | 動作 |
|------|------|
| `Space + \` | 縦に分割 |
| `Space + -` | 横に分割 |
| `Ctrl+h/j/k/l` | ウィンドウ間を移動 |

---

## プラグイン操作

### Neo-tree（ファイルツリー）

起動時に自動でフォーカスされる。

| キー | 動作 |
|------|------|
| `Space + e` | ファイルツリーをトグル |
| `Enter` | ファイルを開く |
| `a` | ファイル/ディレクトリを作成 |
| `d` | 削除 |
| `r` | リネーム |
| `y` | コピー |
| `x` | カット |
| `p` | ペースト |
| `H` | 隠しファイルの表示/非表示 |

### gitsigns（Git 差分）

| キー | 動作 |
|------|------|
| `Space + g + b` | 現在行の blame を表示 |
| `Space + g + p` | 差分をプレビュー |

### Trouble（エラー一覧）

| キー | 動作 |
|------|------|
| `Space + x + x` | エラー/警告一覧を開く |
| `Space + x + d` | ファイルの診断を表示 |

## Ghostty ペイン移動

`gdev` 起動後のペイン間移動。

| キー | 動作 |
|------|------|
| `Cmd+h` | 左ペイン（Nvim）へ |
| `Cmd+l` | 右ペイン（Claude Code）へ |

## よく使うコマンド

| コマンド | 動作 |
|----------|------|
| `:w` | 保存 |
| `:q` | 閉じる |
| `:wq` | 保存して閉じる |
| `:q!` | 保存せず強制終了 |
| `:e {ファイル}` | ファイルを開く |
| `:Lazy` | プラグイン管理画面を開く |
| `:Mason` | LSP/ツール管理画面を開く |
| `:checkhealth` | Neovim の健全性チェック |
