# インスタグラム台本作成スキル

ObsidianのVaultに蓄積されたマーケティングフレームワークを活用して、
視聴者の心理を動かすインスタリール・フィード投稿の台本を作成する。

---

## ステップ 1: ヒアリング

台本作成前に以下を確認する。必要な情報が会話の中にあれば質問をスキップして構わない。

**確認事項**（未指定のものだけ聞く）：

1. **B-Level**：どのBeliefステージの動画か？
   - B-1：共感・問題の理解（Cold → Warm）
   - B-2：他の方法の限界を示す
   - B-3：ユニークなメソッドを見せる（最重要・差別化）
   - B-4：オファーの価値と購入の決断
   - B-5：理想の未来のビジョン
   - 指定なし：テーマから最適なBを提案する

2. **動画の長さ**：15秒 / 30秒 / 60秒 / 90秒（デフォルト：60秒）

3. **テーマ・伝えたいメッセージ**：今回の動画で伝えたいことは何か？

4. **ターゲット**：誰に向けた動画か？（悩み・職業・状況）

5. **Proof タイプ**（Claimの裏付け方）：
   - Pathos（感情・ストーリー）
   - Ethos（権威・データ・実績）
   - Logos（論理・仕組み・因果関係）

## ステップ 3: 台本設計

### 時間構成

| パート | 時間 | 目的 |
|--------|------|------|
| **フック** | 0〜3秒 | スクロールを止める。「え？」と思わせる |
| **共感 / 問題提示** | 3〜20秒 | Movie Sceneで「自分ごと」にさせる |
| **核心メッセージ** | 20〜50秒 | Claim + Proof で納得させる |
| **CTA** | 最後5〜10秒 | 次の行動を明確に促す |

60秒以下の動画は比率を維持しながら各パートを短縮する。

### B-Level別の台本の焦点

**B-1（共感）**
- ネガティブMovie Sceneから始める（顧客が「これ自分だ」と思える具体的な場面）
- 「あなたは悪くない」「誰も教えてくれなかっただけ」を伝える
- 仮想敵（社会・固定観念・業界常識）に責任を向ける
- フレームワーク：PAS または PSONA が最適

**B-2（他の方法の限界）**
- 視聴者が試したであろう方法を3つ列挙してから否定する
- 一時的な効果は認める（全否定しない）
- 根本原因の指摘で締める
- フレームワーク：AIDA または ACCA が最適

**B-3（ユニークなメソッド）**
- Aha Momentを演出する（「なるほど！」の瞬間）
- メソッドに名前をつけて記憶に残す
- 図解・比較をテロップで見せる
- B-2との接続を明確にする（「だからこれが必要」）
- フレームワーク：QUEST または GDT が最適

**B-4（オファーの価値）**
- 受講生・顧客の具体的な成果を数字で見せる
- 価格ではなく価値で語る
- 「今始めないと損」ではなく「未来の自分への投資」として伝える
- フレームワーク：AIDA が最適

**B-5（理想の未来）**
- B-1のネガティブシーンと対になるポジティブMovie Sceneを描く
- 「何をするか」ではなく「誰になるか」のアイデンティティ変化
- 朝・余裕・笑顔・コーヒーなど具体的な感覚的描写
- フレームワーク：PSONA が最適

## ステップ 5: 品質チェック

台本を書いたら以下を確認する：

**視聴者の心理チェック**
- [ ] フックは3秒以内に「え？」を作れているか
- [ ] Movie Sceneは五感を使って具体的に描写されているか
- [ ] 「あなたは悪くない」メッセージが自然に含まれているか
- [ ] Claim（主張）が1文で明確に伝わるか
- [ ] Proof（証拠）がClaimを裏付けているか

**インスタ媒体チェック**
- [ ] テロップの言葉はシンプルで中学生にも分かるか（KISS原則）
- [ ] 動画時間は90秒以内か
- [ ] CTAが1つに絞られているか（フォロー・コメント・リンクのどれか1つ）
- [ ] B-Levelのゴールに沿った内容か

**BTRNUTSSチェック**（タイトル・フックに使う）
- B (Benefit)：ベネフィットが明確か
- T (Trust)：信頼を示す要素があるか
- N (Number)：数字が入っているか
- U (Unique)：他との違いが出ているか
- S (Surprise)：意外性があるか
- S (Story)：物語になっているか

## 失敗モード

- guruポジション（教祖・指南役の目線になる。「同じ悩みを持つ新卒」の目線に戻す）
- 実体験の種がない主張だけの台本
- 1本に複数のBeliefを詰め込む

## 参照フレームワーク早見表

### フレームワーク

| 名前 | 向いている動画 |
|------|--------------|
| **PAS** | B-1共感動画、問題提起中心 |
| **PSONA** | B-1・B-5、共感から行動へ |
| **AIDA** | B-2・B-4、注意引き→欲求→行動 |
| **ACCA** | B-2、論理的に限界を説明する |
| **QUEST** | B-3、ターゲットを絞って教育する |
| **GDT** | フックと短尺動画のタイトル |

### 心理トリガー

| トリガー | 台本での使い方 |
|---------|--------------|
| **返報性** | 無料ノウハウを先に惜しみなく出す |
| **社会的証明** | 「○○人が実践」「受講生の声」 |
| **権威** | 実績・データ・専門家引用をテロップ化 |
| **好奇心の隙間** | フックで答えを出さずに続きを引く |
| **希少性** | 「知ってる人が少ない」（B-3向け） |

### Movie Scene 5要素（感情的共感に必須）

1. **具体的な会話** → テロップで表現
2. **感情状態** → 顔・表情・動作で表現
3. **感覚的描写** → BGM・効果音・映像の質感で表現
4. **時間と場所** → 映像の背景・テロップで表現
5. **内面の思考** → ナレーションで表現
