## 1. 平板化の診断表

知識をそのまま出すと退屈になる4大要因。読者に「なぜ退屈か」を先に見せてから直すと納得感が出る。

| 読者の感覚 | 正体 | 反転 |
|---|---|---|
| 興味がそそられない | リスト構造＝起伏ゼロ。次が予測でき脳が先を読む理由を失う | 一本の物語にし「次どうなる？」を作る |
| 繋がりがない | 読者本人が一人も出てこない。〝人類一般〟の話で自分ごとにならない | 読者を主人公「君」にし、実際の現場を描く |
| 具体アクションがない | ゴールが「覚えろ」止まり。明日いちばんに何をするかが無い | 終幕に「明日やる1個」を置く |
| 何も残らない（致命傷） | 主役が多すぎ（要点を全部同格に並列）＋科学を盛りすぎ | 各幕1点突破＋ヒーロー科学1本に絞る |

> 出典原則：網羅・収集・出典列挙は良い"地図/資料"を生むが、その同じ網羅志向が"読み物"を平板にする。出口では別の筋肉（捨てる・1点に賭ける）が要る。

## 2. 映画の濃度を出す技法

要約（「君は質問で掘った」）でなく、実時間で演じる（実際のセリフの往復＋内心）。

- **内心の声を `（ ）` で実時間で出す**：「（やばい。何か言わなきゃ。面白いことを——）」。沈黙は「一秒。二秒。」と刻む。
- **五感で設定を1つ置く**：場所の音・匂い・手触り（「冷蔵庫の低い唸り」「指先が冷たくなる」）。冒頭の没入を作る。
- **"危ない、寸前で気づく" のテンション**：主人公が一度NGをやりかけ→寸前で正解へ軌道修正する瞬間を見せる（「喉まで『千葉のほうですか？』が出かかった。だが寸前で気づく」）。技法が"生きて"見える最強の演出。
- **相手の微反応で技の手応えを描く**：「目が、ほんの少し、変わる」「手が止まった」「声のトーンが一段下がった」。効いたことを説明せず、相手の体で見せる。
- **セリフは口語のフィラー込み**：「いやあ」「あー、わかる？」「えっと、正直」。整いすぎたセリフは嘘くさい。
- **地の文のリズムを変える**：要所で一行を独立させて「間」を作る（「——来た。」「入った。」）。
- **教えは"ナレーション"で軽く差す**：物語の声のまま1〜2文で原理に触れ、すぐ場面に戻る。箱（カード）は使わない。箱は幕末だけ。

## 3. 幕の実演テンプレ（骨格）

各幕＝1つの主戦場。下の流れで1場面を最後まで演じる。

```
[設定]   五感で場所と、主人公の逃げ/不安を1段落。相手が登場し逃げ場を断つ。
[トリガー] 相手の何気ない一言。主人公の内心が真っ白になる（実時間で）。
[反転]   主人公が"ヒーローの事実"を思い出し、戦法を切り替える決意（喋る→引き出す等）。
[実演]   具体的な一手→相手の微反応→もう一段→相手のスイッチON、を往復セリフで。
         途中で1回 NGをやりかけ寸前で修正（テンション）。
[小さな代償/障害] 任意。地雷・マウント・沈黙などの障害を1つ入れて処理を見せる。
[ペイオフ] 相手の態度が変わる一言。主人公の内心の気づき（「ほとんど喋っていないのに」）。
[幕末カード] その幕で使った手を [!success] で通し番号つきで畳む。
```

幕の数＝主戦場の数（2〜4が目安）。**テンポを変える**：1幕=スローバーン（1対1）、2幕=速いカット割り（多人数）、3幕=短く温かい（再会・継続）。

最終構成：`導入 → 各幕(物語+幕末カード) → 終幕(動機への回答+持ち帰り) → 明日の実装(表) → カード索引表`。

---

## 4. 二層を両立させる原則

- **物語本文は1点突破**：1幕で扱う主役テクは絞る。残りはカードへ逃がす。
- **カードは幕末のみ**：シーン途中に箱を挟むと没入が切れる。
- **索引表で辞書化**：末尾に全カードの「# / 名前 / 一言」表。試合前30秒で開ける。
- **科学はヒーロー1本＋溶かし**：転換点の1本だけドラマ、他は1行＋括弧出典。詳細出典は元の辞書ノートへリンクで逃がす。

## 4.5 縦スクロール組版（出力の体裁・必須）

内容が良くても、横に長いパラグラフは「流し見」できない。人は横より縦のほうが認知負荷が低い。**縦スクロールで一直線に読み下せる状態**を必須の体裁とする。

- **1文＝原則1行**。長い文は意味の切れ目（読点・接続・「——」）で改行する。
- **意味の塊（1〜3行）＝1ブロック**、ブロック間は**空行で区切る**。「1文1行＋全行空け」にはしない（間延びして縦に無限に伸び、流れが切れる）。塊で呼吸を作る。
- **改行は意味の境界に置く**。文の途中の不自然な位置で折らない。
- **ドラマの一行は独立させる**（「——来た。」「入った。」を前後空行で孤立させると効く）。
- **セリフは1発言1行**。地の文と空行で分ける。
- **カード内も項目ごとに改行**（Calloutは各行頭に `>`、項目間は `>` だけの空行）。
- 目安：スマホ幅で1ブロックが画面を横断しない長さ。迷ったら短く割る。

> 破綻条件：割りすぎると間延びする。"1文1行だが、塊で空けてブロック化" のバランスを守る。

## 5. チェックリスト（出す前）

- [ ] 冒頭3行で読者が「自分の話だ」と感じる（読者本人＋具体シーン）
- [ ] 各幕は1場面・1点突破（主役が増えていない）
- [ ] "危ない→寸前で修正" のテンションが最低1回ある
- [ ] 効果を説明せず相手の微反応で見せている
- [ ] 科学は転換点に1本（盛りすぎていない）
- [ ] カードは幕末のみ・通し番号・索引表あり
- [ ] 終幕に「持ち帰り」と「明日やる1個」
- [ ] テンポが幕ごとに変化している（長尺の中だるみ対策）
- [ ] **縦スクロール組版**：1文1行・意味の塊でブロック化・空行区切り（横長パラグラフが無い）
- [ ] 媒体の住み分けを侵していない（体験談/技術/セールスは別スキル）
